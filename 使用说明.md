# 持有型ABS项目爬虫 v2 使用说明

## 工作流

1. 读取 `data/input/` 中最新的 xlsx 基准文件
2. 分别爬取上交所和深交所的持有型ABS项目
3. 与基准文件比对，标注新增和状态变更
4. 导出含变更标注的Excel到 `data/output/`

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

### 一键运行（推荐）

```bash
python main.py
```

程序会自动完成:
1. 读取基准文件 → 构建项目索引
2. 爬取上交所（Playwright自动化）
3. 爬取深交所（优先Excel下载，失败走API/页面）
4. 与基准比对 → 导出带变更标注的Excel

### 运行参数

| 参数 | 说明 |
|------|------|
| `--headless` | 浏览器无头模式（后台运行） |
| `--exchange sse` | 只爬上交所 |
| `--exchange szse` | 只爬深交所 |

```bash
python main.py --headless              # 后台运行
python main.py --exchange sse          # 只爬上交所
python main.py --exchange szse         # 只爬深交所
```

## 基准文件

放置在 `data/input/` 目录下，格式为 `.xlsx`。
程序会自动读取最新的那个文件。

**必要列**：`债券名称`、`交易所`、`项目状态`、`更新日期`、`受理日期`

## 输出格式

输出到 `data/output/持有型ABS项目汇总_YYYYMMDD.xlsx`

包含3个Sheet:
- **全部项目**: 爬取到的所有持有型ABS项目
- **新增项目**: 基准文件中不存在的项目
- **状态变更**: 基准文件中存在但状态/时间等字段有变化的项目

新增行为 **绿色** 标注，状态变更行为 **黄色** 标注。
